# ブロック崩し

Canvas + Vanilla JS で作られた、3ステージ構成のブロック崩しゲームです。

## 遊び方

`index.html` をブラウザで開くだけで起動します。サーバー不要です。

## 操作方法

| キー | 操作 |
|------|------|
| `←` / `A` | パドルを左に移動 |
| `→` / `D` | パドルを右に移動 |
| `Space` | スタート / 続ける / 次のレベルへ |
| `Esc` | ポーズ / ポーズ解除 |

ポーズ中は `↑↓` または `W` / `S` でメニュー選択、`Space` で決定できます。

## ゲームの流れ

1. タイトル画面でスペースキーを押してスタート
2. ボールを落とさないようにパドルで打ち返し、すべてのブロックを破壊する
3. 3ステージすべてクリアすると **ALL CLEAR!!**

ミスをするとライフが1つ減ります。ライフが0になると **GAME OVER**。

## ステージ概要

### Level 1 〜 2

各レベルにランダムで選ばれる複数のブロック配置があります。レベルが進むほどパドルが小さくなり、ボールのスピードが上がります。

| レベル | パドル幅 | 初期スピード |
|--------|----------|-------------|
| 1 | 80px | 4 |
| 2 | 70px | 5 |
| 3 | 60px | 6 |

### Level 3 — ファイナルボス

ブロックがHPを持ち、複数回ヒットしないと破壊できません。

- **HP 1〜3** — ダメージを受けるとブロックにひびが入る
- **ブロック振動** — 一部の列が左右に揺れる
- **エンレイジ** — 残りHP が40%以下になると激化。ボール・パドルが赤く変わり、スピードが大幅に上昇する
